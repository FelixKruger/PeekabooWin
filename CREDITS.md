# Credits

## Peekaboo Inspiration

PeekabooWin is directly inspired by [Peekaboo](https://github.com/steipete/Peekaboo) by Peter Steinberger.

- Original repository: [steipete/Peekaboo](https://github.com/steipete/Peekaboo)
- Project site: [peekaboo.boo](https://peekaboo.boo)

Peekaboo showed what a serious AI-oriented desktop automation tool can look like: screen understanding, structured automation, and a clean bridge into AI tooling through an MCP server and CLI.

PeekabooWin is a Windows-first rewrite of the practical desktop automation loop for people who want that style of workflow on Windows. It is not an official port and is not affiliated with Peter's project.

## Thanks

Thanks to Peter for open-sourcing Peekaboo and setting a high bar for both product clarity and technical ambition in AI desktop tooling.
